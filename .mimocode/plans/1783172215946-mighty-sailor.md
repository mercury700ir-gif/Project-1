## Architecture
```
Section (full-width)
  └── Column (1-4 per section)
        └── Widget (drag-reorderable)
```

## UI Components
- **Canvas**: Live preview with click-to-select
- **Widget Panel**: Left sidebar with widget library
- **Settings Panel**: Right sidebar with Content/Style/Advanced tabs
- **Navigator**: Tree view of all elements
- **Toolbar**: Undo/Redo, Responsive toggle, Save, Preview
- **History**: Revision list with restore

## Files to Modify
- `admin.html` — New page builder HTML structure
- `admin.js` — Widget system, drag-drop, canvas, settings
- `admin-styles.css` — Complete editor styles
