# پنل مدیریت — طرح جامع بهبودها

## فاز ۱: بنرها و مدیریت تصاویر (Elementor-like)

### ۱.۱ ویرایشگر بنر پیشرفته
- اضافه کردن فیلدها: `object-position` (نقطه کانونی), `object-fit` (scale/cover/contain)
- اسلایدر position-X و position-Y برای تنظیم نقطه نمایش عکس
- پیش‌نمایش زنده بنر با تغییرات
- CSS: `object-position` روی تصویر بنر اعمال شود

### ۱.۲ فایل منیجر برای انتخاب عکس از گالری
- Modal با نمایش شبکه‌ای تصاویر گالری
- کلیک روی عکس → انتخاب → بازگشت آدرس به فیلد بنر
- اطلاعات فایل: نام، سایز، فرمت

## فاز ۲: مدیریت کاربران

### ۲.۱ سیستم نقش‌ها
- سه نقش: `admin` (مدیر سایت), `editor` (مدیر محتوا), `subscriber` (کاربر عادی)
- فیلد نقش در فرم افزودن/ویرایش کاربر
- ذخیره نقش در localStorage

### ۲.۲ تب دوگانه کاربران
- **تب ۱: مدیریت** — مدیر سایت + مدیر محتوا
  - افزودن، ویرایش، حذف
  - نمایش ایمیل، شماره، رمز
  - تغییر رمز عبور
- **تب ۲: اعضای سایت** — کاربران عادی ثبت‌نام‌شده
  - نمایش نام، ایمیل، شماره
  - ویرایش اطلاعات
  - تغییر رمز
  - تیک عضویت خبرنامه

---

## فاز ۳: گالری پیشرفته

### ۳.۱ اطلاعات تصاویر زیر هر عکس
- نام فایل، سایز (px)، فرمت، حجم (KB/MB)
- فیلد ویرایش نام و تگ

### ۳.۲ تبدیل فرمت تصویر
- گزینه‌های تبدیل: SVG, WebP, JPEG, PNG
- توضیحات فارسی هر فرمت:
  - **WebP/JPEG**: عکس‌ها و تصاویر پیچیده — حجم کمتر، کیفیت بالا
  - **PNG**: لوگو، گرافیک، شفافیت — بدون افت کیفیت
  - **SVG**: آیکون‌ها و تصاویر برداری — مقیاس‌پذیر
  - **WebP**: انیمیشن‌ها — پشتیبانی از GIF متحرک
- دکمه تبدیل + نمایش پیش‌نمایش

## فاز ۴: وبلاگ

### ۴.۱ انتشار زمان‌دار
- در جدول پست‌ها: دکمه «انتشار» با دو گزینه
  - **انتشار فوری**: وضعیت → published
  - **انتشار زمان‌دار**: انتخاب تاریخ/ساعت → وضعیت → scheduled
- نمایش وضعیت «در انتظار انتشار» + تاریخ در جدول
- بررسی خودکار scheduled posts (هر ۳۰ ثانیه)

### ۴.۲ ویرایشگر پیشرفته (Quill)
- اضافه کردن دکمه «نمایش کد HTML» → toggle بین WYSIWYG و source
- دکمه درج تصویر از گالری (File Manager modal)
- دکمه درج ویدئو از گالری (File Manager modal)
- فیلد URL ویدئو بالای ادیتور → اتصال به فایل منیجر

### ۴.۳ هوش مصنوعی پیشرفته
- تولید محتوای RTL با عکس‌های مرتبط
- عکس header: انتخاب از گالری با نام مرتبط
- عکس وسط مطلب: انتخاب از گالری
- ذخیره عکس‌ها در گالری با نام مرتبط
- رعایت SEO: عنوان، meta، تگ‌ها
- **پیش‌نمایش**: دکمه Preview → نمایش موقت محتوا
- **انتشار مستقیم**: دکمه انتشار از زیر AI

## فاز ۵: پلاگین‌ها

### ۵.۱ آپلود پلاگین
- دکمه «آپلود فایل پلاگین» → انتخاب فایل zip/rar/7z
- ذخیره در localStorage

### ۵.۲ Plugin Edit Tool
- بخش زیر JSON Manifest برای هر پلاگین
- نمایش کد پلاگین در textarea/code editor
- دکمه ذخیره / رد تغییرات

### ۵.۳ تنظیمات پلاگین
- دکمه تنظیمات → Modal باز شود
- فیلدهای تنظیمات داینامیک
- ذخیره → Modal بسته شود + پیام موفقیت + نمایش سبز زیر پلاگین

### ۵.۴ فایل آموزش پلاگین‌نویسی
- `PLUGIN-GUIDE.html` — صفحه HTML زیبا با RTL
- فهرست مطالب، مراحل ساخت پلاگین، ساختار فایل، کد نمونه

## فاز ۶: شبکه‌های اجتماعی

### ۶.۱ دستورالعمل اتصال
- زیر هر شبکه: مراحل اتصال دستی (مرحله‌به‌مرحله)
- WhatsApp: WhatsApp Business API + Webhook
- Telegram: BotFather + Bot Token
- Bale: ساخت ربات از @BaleBot
- Rubika: پنل توسعه‌دهندگان
- Instagram: Facebook Developer + Graph API

### ۶.۲ تب ربات‌ها
- لیست ربات‌های متصل
- افزودن ربات جدید
- تنظیمات هر ربات

### ۶.۳ زمان‌بندی پیشرفته
- جداسازی اکانت‌های عادی و ربات‌ها
- برنامه انتشار مجزا برای هر کدام
