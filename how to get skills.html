How To Get Skills

راهنمای استفاده از فایل‌های Skill Markdown در هوش‌های مصنوعی مختلف.

Skill چیست؟

Skill یک فایل Markdown است که دستورالعمل تخصصی، workflow، محدودیت‌ها، مثال‌ها و مسیر فایل‌های کمکی را برای یک AI Agent تعریف می‌کند.

ساختار پیشنهادی SKILL.md

---
name: frontend-design
description: Use when building polished production frontend interfaces.
---

# Workflow
1. Read the project structure.
2. Follow the existing design system.
3. Implement the UI.
4. Test responsive behavior.

# Rules
- Keep changes scoped.
- Prefer existing helpers.
- Verify before final response.

استفاده در Codex یا Agentهای فایل‌محور

  • فایل را در مسیر skills یا کنار پروژه قرار دهید.
  • در prompt بنویسید: از skill با نام موردنظر استفاده کن.
  • Agent باید ابتدا فایل Skill را بخواند و فقط بخش‌های مرتبط را اجرا کند.
از skill frontend-design استفاده کن و صفحه مدیریت محتوای پروژه را مطابق دستورالعمل آن طراحی و پیاده‌سازی کن.

استفاده در ChatGPT، Claude، Gemini و مدل‌های مشابه

  • محتوای SKILL.md را در ابتدای گفتگو یا به عنوان attachment بدهید.
  • درخواست کنید که مدل آن را به عنوان دستورالعمل پروژه در نظر بگیرد.
  • برای جلوگیری از شلوغی، فقط Skill مرتبط با همان کار را وارد کنید.
این فایل SKILL.md دستورالعمل انجام کار است. آن را بخوان، خلاصه کن، سپس فقط بر اساس همین workflow تغییرات پروژه را انجام بده.

نکات کیفیت

  • Skill را کوتاه، دقیق و task-oriented بنویسید.
  • قوانین غیرقابل مذاکره را جدا کنید.
  • مثال prompt و مثال خروجی اضافه کنید.
  • اطلاعات محرمانه را داخل Skill قرار ندهید.
