How To Build A Website

راهنمای کامل ساخت سایت شخصی فارسی با پنل مدیریت، backend، blog، gallery، page builder، banners و PopUps.

Prompt اصلی

یک وب‌سایت شخصی فارسی راست‌به‌چپ برای مشاور بازاریابی بساز. سایت شامل صفحه اصلی، درباره من، تماس، وبلاگ، ثبت‌نام، ورود، پنل مدیریت، مدیریت صفحات با page builder، مدیریت محتوا برای بنر و PopUp، گالری، فایل‌ها، کاربران، تنظیمات و backend API باشد. طراحی باید حرفه‌ای، سریع، responsive و user friendly باشد.

معماری پیشنهادی

Frontend
HTML, CSS, JavaScript، پنل tab-based، فارسی RTL.
Backend
Express routeها برای auth، posts، pages، banners، media و settings.
Storage
دیتابیس برای داده‌های پایدار و localStorage فقط برای fallback محیط استاتیک.

Prompt طراحی پنل مدیریت

یک پنل مدیریت فارسی شبیه ابزارهای حرفه‌ای CMS بساز. Sidebar، topbar، جدول‌های واکنش‌گرا، فرم‌های قابل فهم، حالت empty، پیام خطا، و عملیات سریع روی هر ردیف داشته باشد. UI باید برای کاربر غیرتخصصی قابل فهم باشد.

Prompt ساخت Page Builder

در پنل مدیریت، بخش مدیریت صفحات بساز. لیست صفحات، ایجاد صفحه، canvas، navigator، پنل تنظیمات، undo/redo، responsive preview و ویجت‌های عنوان، متن، تصویر، ویدئو، دکمه، فرم، CTA، بنر، گالری، جدول قیمت و شبکه پست‌ها را اضافه کن.

Prompt مدیریت محتوا

یک تب مدیریت محتوا بساز که شامل بنرها، PopUps و آنالیتیکز باشد. هر آیتم title، content، CTA، display position، targeting rule، trigger، schedule، active state، views، conversions و conversion rate داشته باشد.

Prompt تست

همه مسیرهای UI و backend را تست کن: ورود، تغییر تب‌ها، ساخت صفحه، ذخیره builder، مدیریت بنر، مدیریت PopUp، ویرایش گالری، جدول وبلاگ، تغییر وضعیت پست، responsive موبایل و خطاهای JavaScript.
