۱. نمای کلی پروژه

سایت شخصی «مهبد نادری» شامل دو بخش اصلی است: سایت عمومی (فرانت‌اند) و پنل مدیریت با بک‌اند کامل. تمام داده‌ها در پایگاه داده MySQL ذخیره می‌شوند و از طریق API RESTful قابل دسترسی هستند.

پرامپت اولیه

یک سایت شخصی حرفه‌ای برای «مهبد نادری» (معمار رشد دیجیتال و مشاور بازاریابی) با بک‌اند Node.js + Express + MySQL بساز. سایت شامل صفحات عمومی (خانه، درباره، تماس، نوشته‌ها، ثبت‌نام) و پنل مدیریت کامل با ۹ تب باشد. احراز هویت با JWT و bcrypt باشد. تمام صفحات RTL و فارسی و فونت IRANSans باشد.

۴. معماری بک‌اند

پرامپت بک‌اند

بک‌اند Node.js با Express بساز. احراز هویت JWT + bcrypt باشد. پایگاه داده MySQL با ۱۵ جدول. فایل‌ها با multer آپلود شوند. تمام endpointها با CORS فعال باشند. فرانت‌اند از مسیر اصلی سرو شود.

۱۰. اتصال فرانت‌اند به بک‌اند

ماژول API Client (api.js)

فایل api.js یک wrapper مرکزی برای تمام درخواست‌های fetch است. توکن JWT در localStorage ذخیره شده و در هدر هر درخواست ارسال می‌شود.

const API_BASE = window.location.origin + '/api';
const api = {
  token: localStorage.getItem('api_token'),
  async request(method, path, body) {
    const headers = { 'Content-Type': 'application/json' };
    if (this.token) headers['Authorization'] = 'Bearer ' + this.token;
    const opts = { method, headers };
    if (body && method !== 'GET') opts.body = JSON.stringify(body);
    const res = await fetch(API_BASE + path, opts);
    const data = await res.json();
    if (!res.ok) throw new Error(data.error);
    return data;
  },
  login(e, p) { return this.request('POST', '/auth/login', {email:e, password:p}); },
  register(n, e, ph, p) { return this.request('POST', '/auth/register', {name:n, email:e, phone:ph, password:p}); },
  // ... تمام متدهای CRUD
};

اتصال هر فایل

  • admin.html: <script src="api.js"> + <script src="admin.js">
  • register.html: ورود/ثبت‌نام از طریق api.login() و api.register()
  • reset-password.html: بازیابی رمز از طریق api.resetPassword()
  • blog.html: خواندن پست‌ها از api.getPosts()
  • تمام صفحات عمومی: api.js برای بررسی وضعیت لاگین
